# PHP Snippets for Visual Studio Code

A collection of custom PHP snippets that i use in Visual Studio Code to speed up development.

## Installation

1. Open Visual Studio Code.
2. Press `Ctrl+Shift+P` (`Cmd+Shift+P` on macOS).
3. Select **Snippets: Configure Snippets**.
4. Choose **PHP**.
5. Replace or merge the contents of `php.json` with the snippets from this repository.

## Usage

Type a snippet prefix and press `Tab` (or `Enter`, depending on your VS Code settings) to expand it.

## Available Snippets

| Prefix | Description |
|---------|-------------|
| `pubf` | New public function |
| `prof` | New protected function |
| `prif` | New private function |
| `class` | New class |
